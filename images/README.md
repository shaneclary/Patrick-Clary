# Photographs

Drop image files here (`.jpg`, `.png`, `.webp`), then register each one in
`js/content.js` under `gallery`:

```js
{ src: "images/patrick-walking.jpg", caption: "Walking the shore road, autumn." },
```

Guidance for choosing photographs — curated, not a photo dump:

- Documentary, not posed. Patrick listening, walking, teaching.
- Natural light. Shallow depth of field where possible.
- Hospital hallways, nature, hands, patients' artwork, conferences, family.
- Portrait orientation (4:5) fits the frames best, but any shape works.
